# Minesweeper Game with Undo function in Java

Minesweeper is a puzzle game where the player clears a board without detonating hidden mines. Players click on fields to uncover them and can mark suspected mines. It's popular on Windows since 1990.

However, it is just a basic game, so we decide to make a slight difference which is the undo function. By request this function, the players are able to recall the current move and continue the game.
